# flower-classifier-keras
flower classifier use VGG19 transfer learning ,pretrained model.

#  How to do it
1.下载Flower  | Kaggle

2.解压找到文件夹存放

3.修改代码中base_path = '改成存放数据集的地址'

4.训练得到结果

5.之后可以修改代码如：def create_model_from_VGG19()的部分改成其他比较好的模型，

替换数据集做自己想要的输入
